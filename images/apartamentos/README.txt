==============================================
INSTRUCCIONES PARA AGREGAR IMÁGENES DE APARTAMENTOS
==============================================

Carpeta creada: images/apartamentos/

Estructura de carpetas:
- images/apartamentos/apartamento_0/ (Para el primer apartamento)
- images/apartamentos/apartamento_1/ (Para el segundo apartamento)
- images/apartamentos/apartamento_2/ (Para el tercer apartamento)
- images/apartamentos/apartamento_3/ (Para el cuarto apartamento)

Para agregar imágenes de los apartamentos, sigue estos pasos:

1. Copia tus imágenes a la carpeta correspondiente:
   - images/apartamentos/apartamento_0/ (Propiedad 0 - Eucalipto)
   - images/apartamentos/apartamento_1/ (Propiedad 1 - Penthouse)
   - images/apartamentos/apartamento_2/ (Propiedad 2 - Apartamento Familiar)
   - images/apartamentos/apartamento_3/ (Propiedad 3 - Estudio Moderno)

2. Nombra tus imágenes dentro de cada carpeta:
   - imagen_1.jpg (o .png, .webp)
   - imagen_2.jpg
   - Etc.

3. Formatos soportados: .jpg, .jpeg, .png, .webp

4. Cantidad recomendada por apartamento: 4-6 imágenes

5. Para usar tus imágenes, modifica propiedad.html y cambia 
   las URLs externas por las rutas locales, por ejemplo:
   "images/apartamentos/apartamento_0/imagen_1.jpg"
   
   Busca la sección "images:" en el script de propiedad.html
   y reemplaza las URLs de Unsplash por las rutas de tus imágenes locales.
